# Agriculture Farming Monitoring Website

## Overview

The Agriculture Farming Monitoring Website is a web application developed using Python Flask, HTML, CSS, and JavaScript. It helps farmers access important agricultural information through a simple and user-friendly interface.

## Features

- User Login and Signup
- Farming Information
- Crop Health Monitoring
- Market Information
- Dashboard
- Responsive Design

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
